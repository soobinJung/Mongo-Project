# Notice Project

### MongoDB 란

MongoDB는 크로스 플랫폼, 문서 지향 데이터베이스 시스템으로, NoSQL 범주에 속합니다. 이는 전통적인 테이블 기반의 관계형 데이터베이스 대신, JSON과 유사한 동적 스키마 형식의 문서(이를 BSON이라 칭함)를 사용하여 데이터를 저장합니다. 이 접근 방식은 특정 애플리케이션에서 데이터 통합을 더욱 용이하고 빠르게 만들어주는 장점을 가집니다.
